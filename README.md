# wf-themes

Firefox extension that themes a fixed set of websites (Discord, Claude, GitHub,
Reddit, Teams, MTools) and reacts in real time to the active theme published by
the [wmenu](https://github.com/YannickHerrero/wmenu) global theme selector.

Pick a theme in wmenu → matching CSS is applied across all open tabs within a
few hundred milliseconds.

Detailed install and architecture docs are in [a later commit] — this is a stub.
